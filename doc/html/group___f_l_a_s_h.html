◆ hal_flash_erase()

int hal_flash_erase ( const FlashDevice *  device,
PartitionId  in_partition,
uint32_t  offset,
uint32_t  size 
)

Erase an area on a Flash logical partition.

Note
Erase on an address will erase all data on a sector that the address is belonged to, this function does not save data that beyond the address area but in the affected sector, the data will be lost.
Parameters
[in]deviceThe instance of flash device
[in]in_partitionThe target flash logical partition which should be erased
[in]offsetStart address of the erased flash area
[in]sizeSize of the erased flash area
Returns
Return result
Return values
VSD_SUCCESSOn success, otherwise is error
