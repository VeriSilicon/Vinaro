◆ hal_ppg_read_sample()

int hal_ppg_read_sample ( PpgDevice *  ppg_device,
uint8_t *  red,
uint8_t *  ir,
uint8_t *  green,
uint16_t  exp_size,
int  sample_format,
uint16_t *  act_size 
)

ppg hal API to read sample into 3 buffer in diff sample format

Note
  • the exp_size should be no larger than buffer size;
  • exp_size = (size of one sample in byte) * (expected reading sample numbers with one light type). If not NULL,red,ir,green buffer have the same exp_size
  • act_size: If not NULL,red,ir,green buffer have the same act_size
Parameters
ppg_deviceThe ppg sensor device
redbuffer to store red light data
irbuffer to store ir light data
greenbuffer to store green light data
exp_sizeexpected to read data from sensor in byte
sample_formatsample format in buffer, definition
See also
e_ppg_sample_format
Parameters
act_sizethe actual length of samples in byte which read from sensor and stored in the buffer
Returns
Return result
Return values
VSD_SUCCESSfor succeed, others for failure

◆ hal_ppg_read_sample_bytes()

int hal_ppg_read_sample_bytes ( PpgDevice *  ppg_device,
uint8_t *  samples,
uint16_t  exp_size,
int  sample_format,
uint16_t *  act_size 
)

ppg hal API to mixed one buffer in diff sample format

Note
  • data format likes: RED,IR,RED,IR for RED+IR mode;
  • the exp_size should be no larger than buffer size;
  • exp_size = (enabled LED number) * (size of one sample in byte) * (expected reading sample numbers)

exp_sizeexpected to read data from sensor in byte
sample_formatsample format in buffer, definition

◆ hal_ppg_set_fifo_watermark()

int hal_ppg_set_fifo_watermark ( PpgDevice *  ppg_device,
uint8_t  watermark 
)

ppg hal API to set fifo watermark

Note
this function decide how many new samples can be written to the FIFO before the interrupt is asserted. For example, if watermar set to 0xF, fifo size is 0x20, the interrupt triggers when there is 15 empty space left (17 data samples), and so on.
Parameters
ppg_deviceThe ppg sensor device
watermarkfifo watermark to be set
Returns
Return result
Return values
VSD_SUCCESSfor succeed, others for failure
